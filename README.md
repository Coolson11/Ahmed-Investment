# Creative Agency Website

A visually stunning, high-end website for a creative agency specializing in photography, videography, and graphic design.

## Features

- **Modern Design**: Minimalist yet bold with luxury aesthetic
- **Dark/Light Theme**: Smooth transitions between themes
- **Responsive**: Fully responsive across all devices
- **Animations**: Subtle animations and micro-interactions throughout
- **Portfolio Gallery**: Interactive gallery with lightbox viewing
- **Contact Form**: Modern form with validation
- **SEO Friendly**: Optimized for search engines

## Tech Stack

- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Forms**: React Hook Form with Zod validation
- **Lightbox**: Yet Another React Lightbox
- **Theme**: Next Themes

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `app/`: Next.js app directory with pages
- `components/`: Reusable React components
- `lib/`: Utility functions
- `public/`: Static assets

## Pages

- **Home**: Hero section, services preview, portfolio highlights, testimonials
- **Portfolio**: Categorized gallery with filter and lightbox
- **About**: Company story and team information
- **Services**: Detailed service offerings with pricing
- **Contact**: Contact form, information, and social links

## Customization

- Replace placeholder images in `public/` with actual assets
- Update content in page components
- Modify styles in `globals.css` or component files
- Add more portfolio items in the respective page files

## Deployment

Deploy to Vercel, Netlify, or any platform supporting Next.js.
